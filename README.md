## Networks by Region
  
### Latin America
- [Altermundi](https://altermundi.net/) - Community network project in Quintana, Argentina, focused on connecting rural areas through mesh technology and local governance.
- [CARC](https://carc.libre.org.ar/) - Community network cooperative in Argentina promoting free telecommunications and digital rights.
- [Coolab](https://coolab.org/) - Brazilian cooperative laboratory for community networks and digital culture.
- ElValleReinicia - Community network initiative in Las Calles, Córdoba, Argentina.
- [MonteNet](https://montelibre.net/) - Community-owned network serving Molinari, Argentina.
- NonoLibre - Community mesh network in Nono, Traslasierra Valley, Argentina.
- [Rede Mocambos](https://mocambos.net/) - Network of quilombola communities in Brazil using technology for cultural preservation and communication.

### Europe
- [Antenie](https://antennine.noblogs.org/) - Community wireless network project in Italy.
- [Freifunk](https://freifunk.net/) - Decentralized community mesh network initiative across Germany with hundreds of local groups.
- [Guifi.net](https://guifi.net/) - One of the world's largest community networks, based in Catalonia, Spain, with over 35,000 active nodes.
- [Ninux](http://wiki.ninux.org/) - Wireless community network spanning multiple cities across Italy.

### Africa
- [TunapandaNET](https://tunapanda.org/) - Community network and learning platform in Kibera, Kenya, providing connectivity and digital skills training.
- [Zenzeleni](https://zenzeleni.net/) - Community-owned telecommunications cooperative in rural South Africa.

### Asia
- [Rhizomatica](https://rhizomatica.org/) - Community cellular network initiative in the Philippines and other regions, enabling communities to build their own telecommunications infrastructure.

## Tools & Software

- [gluon](https://gluon.readthedocs.io/en/latest/) - Framework for building custom OpenWrt-based firmware for mesh networks, used by Freifunk communities.
- [LibreMesh](https://libremesh.org/) - Open-source firmware solution for creating and managing wireless mesh networks with minimal configuration.
- [Meshenger](https://github.com/dakhnod/Meshenger) - Android messaging app that works over local WiFi mesh networks without internet connectivity.
- [Meshtastic](https://meshtastic.org/) - Open-source LoRa-based mesh communication platform for off-grid messaging and GPS tracking.
- [OpenWRT](https://openwrt.org/) - Linux-based open-source firmware for embedded devices, widely used as the foundation for mesh network routers.
- [Qaul](https://github.com/qaul/qaul.net) - Decentralized communication application for mesh networks enabling file sharing and messaging.
- [Reticulum](https://reticulum.network/) - Cryptography-based networking stack for building resilient mesh networks over diverse physical mediums.
- [Retroshare](https://retroshare.cc/) - Decentralized communication platform supporting encrypted chat, forums, and file sharing over mesh networks.
- [Yggdrasil](https://yggdrasil-network.github.io/) - Experimental end-to-end encrypted IPv6 overlay network with automatic routing.

## Hardware Projects

- [LibreRouter](https://librerouter.org/) - Open-source hardware router specifically designed for community networks with dual-band support and LibreMesh integration.
- [Mesh Potato](https://www.villagetelco.org/) - Low-cost mesh telephony device combining WiFi mesh networking with VoIP capabilities for rural areas.
- [Meshbox](https://github.com/libremesh/lime-packages) - Plug-and-play mesh network router hardware running LibreMesh firmware.
- [OpenWrt-One](https://openwrt.org/toh/openwrt/one) - Reference hardware platform designed specifically for OpenWrt development and deployment.

## Governance & Policy
- [Community Network Manual by APC](https://www.apc.org/en/pubs/community-networks-manual) - Comprehensive guide by the Association for Progressive Communications on building and sustaining community networks.
- [Community Networks as a public policy](https://altermundi.net/wp-content/uploads/2023/04/NodoTAU_AlterMundi_-2-A-public-policy-in-motion.pdf) - Research paper by Altermundi on community networks as public policy initiatives in Argentina.
- [IGF Dynamic Coalition on Community Connectivity](https://comconnectivity.org/) - Internet Governance Forum initiative promoting community-driven approaches to connectivity and digital inclusion.
