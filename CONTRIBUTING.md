# Contributing

Contributions are welcome! This is a curated list of community networks, mesh technologies, and grassroots connectivity efforts.

## How to Contribute

1. Fork the repository
2. Create a new branch for your changes
3. Make your additions to the appropriate section in `README.md`
4. Open a pull request

## Guidelines

- **Networks** should be community-owned or community-governed connectivity projects. Include the name, country/city, and a link to the project's website.
- **Tools & Software** should be actively maintained open-source projects relevant to community networking, mesh routing, or decentralized connectivity.
- **Hardware Projects** should be open-hardware initiatives designed for community network deployments.
- **Links** must point to the project's official website or source repository.
- **Alphabetical order** within each section is preferred.
- Keep descriptions brief (one line).

## Formatting

Follow the existing format:

```
- [Project Name - Country - City](https://example.com/) - Brief optional description
```

## Reporting Issues

If you find a broken link or an inactive project, please open an issue.
